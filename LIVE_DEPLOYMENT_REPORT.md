## 🛠️ Troubleshooting

### Site Not Loading
**Check**: 
1. GitHub Pages enabled in Settings → Pages
2. Source set to `main` branch, root directory
3. Build status in Actions tab (green = success)
4. Browser cache cleared (Ctrl+Shift+Del)

### Images Not Showing
**Check**:
1. Image files exist in assets/images/
2. File paths are relative (not absolute)
3. Check DevTools Console (F12) for 404 errors
4. GitHub Actions build succeeded

### Styles Not Applied
**Check**:
1. assets/css/style.css is committed
2. style.css path in HTML is correct
3. Clear browser cache
4. Check DevTools Network tab for 404s

### WhatsApp Links Not Working
